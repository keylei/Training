## 项目运行流程

```
1. 生成2个行政,3个工程师和4个销售,将每个员工添加至相应部门的员工列表中,将每个部门添加至公司的部门列表中
2. 打印出所有员工的薪水,所有部门的总薪水,及所有员工的总薪水
3. 按性别打印出所有员工的薪水
4. 移除公司某一部门的经理,并新创建一个新经理加入
5. 打印出当前工资最高的员工

```
