## 目录树
文件结构如下：  

```
3DAOI_grace/
|__README.md                            //项目说明
|__debug                                //程序编译所在的目录
|  |__...                               //编译生成的其他文件
|__src
   |__app                               //数据生成类相关文件
   |  |__...  
   |__job                               //基板以及检测对象类相关文件
   |  |__...  
   |__sdk                               //数据随机生成，形状，异常等相关类文件 
   |  |__...                        
   |__main.cpp                          //程序入口 
   |__.pro                              //用于配置qmake的文件

```

## 程序配置
编译目录设置为../debug(已经设置，如果不行则重新设置为此目录)

